## @yellowryan/tsconfig
> A personal [tsconfig](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) share

#### Install

```sh
npm install -D @yellowryan/tsconfig
```

#### Usage

`tsconfig`
```json
  {
    "extends": "@yellowryan/tsconfig",
    "compilerOptions": {
      "outDir": "dist"
    }
  }
```
